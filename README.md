# 2 Editar um catalogo
### Na tela de edição você pode:
- Editar o nome do catalogo
- Editar o nome da cesta
- Editar o preço da cesta
- Editar a imagem da cesta
- Editar os itens da cesta( Remover ou adicionar novos )
